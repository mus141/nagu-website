## How to use this system

- **Link one stylesheet:** `styles.css` (root). It `@import`s every token + `@font-face`
  (fonts live in `assets/fonts/`). All design values are CSS custom properties — never hardcode
  hexes; use the tokens (`var(--nagu-red)`, `var(--font-display)`, `var(--space-5)`, …).
- **Components** live in `components/<group>/<Name>.jsx` (+ `.d.ts` props + `.prompt.md` usage).
  They are self-contained React, style only via CSS vars, import only React + siblings.
  Available: `Button, Tag, Badge, PriceTag, SectionHeading` (core) · `Input, QtyStepper` (forms)
  · `Card, ProductCard, CharacterCard` (content). Read each `.prompt.md` for API + examples.
- **`_ds_bundle.js`** is a prebuilt UMD bundle of all components, exposed on
  `window.NAGUDesignSystem_56f253`. The card/kit HTML loads it via `<script src=".../_ds_bundle.js">`.
  (It was generated by the authoring environment; regenerate or replace it if you move to a real
  build — e.g. import the `.jsx` directly in your bundler.)
- **UI kits** (`ui_kits/`) are full-screen recreations to copy from:
  `ordering-app/` (mobile takeaway: menu → item sheet → cart) and
  `website/` (desktop drop-style landing built from `uploads/Nagu-Website-Sitemap.md`).
  `ui_kits/ordering-app/menu-data.js` holds the canonical menu (`window.NAGU_MENU`).

## When building artifacts
- Copy the assets you need out of `assets/` (logos, fonts, mascots, product shots, textures).
  **Never redraw logos/mascots or generate new brand art** — use the real files.
- For React: load React 18 + Babel, then `_ds_bundle.js`, then read components off the namespace.
  For production: import the `.jsx` components into your bundler and ship `styles.css`.

## Brand quick-card (non-negotiables)

- **Color:** NAGU Red `#B70D2A` · Near Black `#1E1E1E` · Off-White `#F2F2F2` · Cream `#DCD2C8`.
  Rare premium accent: gold `#C8A24A`. High contrast always. (The website kit uses a slightly
  punchier red `#C8102E` for neon-poster energy — both are on-brand.)
- **Type:** `Nagu Display` (distressed condensed caps — every headline/shout), `Nagu Sans`
  (clean geometric — body/UI), `Nagu JP` (angular katakana/kanji — *small, minimal accents only*).
  Rule: **English big, Japanese small.** Never crowd three languages in one layout.
- **Signature device:** hard black keyline (2–3px) + hard, no-blur offset block shadow
  (`4px 4px 0 #000`). Square corners. Street poster, not soft SaaS.
- **Voice:** short, stacked, three-beat staccato. ALL-CAPS shouts, sentence-case body.
  No emoji. Confident, culturally fluent, never corporate.
- **Motion:** snappy 120–200ms `cubic-bezier(.2,.8,.2,1)`; press "stamps down". No floats/bounce.
- **Backgrounds:** poster black, warm paper/cream, or cinematic neon-Tokyo illustration. Grunge
  texture + torn-paper edges (SVG `feTurbulence`+`feDisplacementMap`) give the street-collage feel.
- **Mascots:** Nagu · Biku · Fifu (sibling samurai-anime, streetwear). Central IP — use as
  avatars, collectibles, manga cutouts.
- **Icons:** Lucide (CDN), 2px stroke, `currentColor`. No bespoke icon set existed. Note: Lucide
  dropped brand glyphs (instagram/twitter) — use `at-sign` / generic stand-ins.
- **NEVER:** childish cartoon overload · cheap/low-status · crowded 3-language layouts · corporate
  · soft SaaS rounding · bluish-purple gradients · emoji.
