Collectible mascot card for the NAGU character IP (Nagu / Biku / Fifu) — framed portrait, display-caps name with a red tail, uppercase role line, optional JP accent, and a red collectible-number stamp. Use for loyalty, "web verse," and brand-IP sections.

```jsx
<CharacterCard
  image="assets/characters/biku.jpeg"
  name="BIKU"
  role="The Culture Creator"
  jp="ビク"
  number={1}
/>
```

The last two letters of `name` automatically color red to echo the wordmark.
