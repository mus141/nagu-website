Small uppercase pill chip — menu categories, sauce/flavor labels, "limited drop" flags, and dual-language JP accents.

```jsx
<Tag tone="red">New Drop</Tag>
<Tag tone="outline">Spicy</Tag>
<Tag jp tone="ink">大胆な味</Tag>
```

Tones: `ink | red | outline | cream | gold`. Set `jp` to render with the Japanese accent face for decorative katakana/kanji.
