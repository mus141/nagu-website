Poster-grade section opener — red uppercase eyebrow, stacked distressed-caps display headline, optional Japanese accent line, and a skewed red underline bar. Use to open any content section, menu group, or hero block.

```jsx
<SectionHeading eyebrow="The Menu" jpAccent="メニュー">Focused. Craveable. Fast.</SectionHeading>
<SectionHeading tone="light" align="center">Bold Flavor. Built Different.</SectionHeading>
```

`tone`: `ink` (dark text on light) or `light` (off-white on dark). `align`: `left | center | right`. Set `underline={false}` to drop the red rule.
